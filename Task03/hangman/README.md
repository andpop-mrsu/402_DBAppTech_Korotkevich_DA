# Игра "Виселица" (Hangman)

Консольная реализация классической игры "Виселица" на PHP с использованием Composer.

## Требования к окружению

- **PHP 7.4** или выше
- **Composer** для управления зависимостями
- Расширения PHP:
  - PDO_SQLite (для работы с базой данных - будет в следующих работах)
  - mbstring (для работы с многобайтовыми строками)

Описание игры
Компьютер загадывает слово из 6 букв. Игрок пытается угадать слово, называя по одной букве.
За каждую неправильную букву рисуется часть виселицы. Игрок побеждает, если угадывает слово до того,
как рисунок виселицы будет завершен.

Игра поддерживает несколько режимов:

-n, --new — новая игра (по умолчанию).

-l, --list — вывод списка всех игр (на данном этапе база данных не подключена).

-r, --replay — повтор игры с указанным ID (пока недоступно).

-h, --help — вывод справки.

Примеры:

./bin/hangman -n # новая игра ./bin/hangman --help # справка

Разработка
Для разработчиков:

Установите зависимости: composer install

Запустите игру: php bin/hangman

Для обновления автозагрузчика: composer dump-autoload -o

Для добавления новых зависимостей: composer require package/name
